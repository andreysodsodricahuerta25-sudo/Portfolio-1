This program is the Daily Sleep Calculator that will enable the users to understand the hours that their night-time hobbies and other activities take away from them. In the input, the user will insert his/her name, number of hours in a day, and the hours used in each of their own individual hobbies. The user can choose any hobby they wish for, for instance, doom scrolling, studying, eating, cleaning the house, playing basketball, reading books, exercising, talking to friends, playing chess, and watching movies. The program then calculates the number of hours taken up by all these activities and determines the remaining number of hours for sleeping.
